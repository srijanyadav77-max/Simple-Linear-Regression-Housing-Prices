# Simple Linear Regression on Housing Prices

## Project Overview
This project builds a Simple Linear Regression model to predict housing prices
using the California Housing dataset from scikit-learn.

## Technologies Used
- Python 3
- Google Colab
- NumPy, Pandas
- Scikit-learn
- Matplotlib, Seaborn

## Steps Performed
1. Loaded the California Housing dataset
2. Performed Exploratory Data Analysis (EDA)
3. Selected features based on correlation
4. Normalized data using StandardScaler
5. Split data into 80% train / 20% test
6. Built and trained a Linear Regression model
7. Evaluated using MAE, MSE, RMSE, and R² score
8. Visualized Actual vs Predicted and Residual plots

## Results
| Metric | Value |
|--------|-------|
| MAE    | ~0.53 |
| RMSE   | ~0.73 |
| R²     | ~0.61 |

## How to Run
Open the `.ipynb` file directly in Google Colab by clicking the file
and selecting "Open in Colab".
